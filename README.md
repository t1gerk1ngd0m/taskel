# README

# アプリ作成の目的
このアプリはmofmof社での研修課題として作成したものです。
このアプリの作成を通して環境構築、Ruby on Railsの基本的操作、Gitの使い方、テスト、Heroku/AWSへのデプロイなどアプリケーション作成における一連の流れを学習しています。

# アプリ機能

# デプロイ方法
ターミナル上で下記のコマンドを実行  
` git push heroku master `  
下記のコマンドを実行し本番環境を確認  
` heroku open `  
本番環境でエラー発生時は下記コマンドで確認  
` heroku logs `
